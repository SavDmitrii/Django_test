Создать виртуальное окружение - python3 -m venv (virt - имя окружения)
Активация виртуального окружения - source virt/bin/activate 
Создать проект Джанго - django-admin startproject (testproject - имя прооекта)
Запуск локального сервера - ./manage.py runserver
Создание приложения Джанго - ./manage.py startapp (myapp - имя приложения)

GIT

Следить за папкой(инизиализировать локальный репозиторий) - git init
Добавить все файлы в локальный репозиторий - git add .
Новый коммит - git commit -m 'first commit'
Список всех коммитов - git log
Сокращеный список коммитов - git log --oneline
Откат к нужному коммиту - git reset --hard (номер коммита)
Соеденить локальный и удаленый репозиторий - ссылка на сайте ГитХаб
Создание ветки в удаленном репозитории - git branch -M main
Перенести файлы из локального на удаленный репозиторий - git push -u origin main
Скачать из удаленного репозитория - git pull origin main
